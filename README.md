# Personal Configuration for Mac and Ubuntu
---------------------------------

## Installation
```
git clone git@github.com:xlpiao/.setup.git
cd .setup
./install.sh
```

## Git Configuration 
```
git config --global user.email "Your Email Address"
git config --global user.name "Your Name"
```
